# OTPGenerateApp
A simple and responsive OTP (One-Time Password) verification system built using *HTML, CSS (Bootstrap), and JavaScript*.

---

## 🚀 Features

- ✅ Generate 4-digit OTP
- ✅ Verify entered OTP
- ✅ Auto-clear input fields on new OTP generation
- ✅ Auto-focus on next input box
- ✅ Success & error messages
- ✅ Responsive UI using Bootstrap
- ✅ Clean and modern design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (Vanilla JS)

---

---

## ⚙️ How It Works

1. Click on *Generate OTP*
2. A random 4-digit OTP will be shown (via alert)
3. Enter the OTP in input boxes
4. Click *Verify OTP*
5. You will see:
   - ✅ Success message (if correct)
   - ❌ Error message (if incorrect)
